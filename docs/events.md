# Events

Package dispatches lifecycle events for teams and membership changes.

## Team Events

- `TeamCreating`
- `TeamCreated`
- `TeamUpdating`
- `TeamUpdated`
- `TeamDeleted`

## Member Events

- `TeamMemberAdding`
- `TeamMemberAdded`
- `TeamMemberRemoving`
- `TeamMemberRemoved`
- `TeamMemberUpdated`
- `TeamMemberInviting`
- `TeamMemberInvited`

Use listeners/subscribers to add audit logging, notifications, or external sync logic.
